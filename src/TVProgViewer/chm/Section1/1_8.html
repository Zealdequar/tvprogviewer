Контекстное меню TVProgViewer

В приложении TVProgViewer два вида контестного меню: для телепрограммы и для значка в трее.

Контестное меню для телепрограммы можно вызвать правым щелчком мыши на таблице с телепрограммой. Доступны комнады для вызова поиска по названию передачи, добавления передачи в классификаторы (жанров и/или к любимым), вызова свойств телеканала, копирования времени и названия передачи в буфер обмена, изимение рейтинга и/или типа.

Контекстное меню для значка в трее Windows позволяет управлять напоминателем, открыть основное окно приложения или выйти из приложения.
