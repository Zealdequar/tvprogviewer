Панели инструментов TVProgViewer

В приложении имеется три разных панели инструментов: одна основная и две другие для фильтрации по жанрам и рейтингам.

Основная панель инструментов содержит команды для управления приложением и состоит из кнопок:

  • «Загрузить» — осуществляет загрузку телепрограммы из интернета.
  • «Экспорт в Excel» — осуществляет выгрузку данных о телепрограмме в пакет MS Excel для дальнейшей обработки и вывода на печать.
  • «Каналы» — открывает редактор каналов.
  • «Поиск» — открывает окно для поиска телепередач.
  • «Жанры» — открывает редактор жанров.
  • «Рейтинги» — открывает редактор рейтингов.
  • «ТВ-тюнер» — открывает окно для работы с тв-тюнером (для активизации кнопки нужно в общих настройках выбрать параметры работы тв-тюнера).
  • «Общие настройки» — открывает общие для приложения настройки.

Некоторые команды основной панели инструментов продублированы в основном меню.
